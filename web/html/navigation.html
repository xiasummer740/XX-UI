二、实用技巧


  1. 如何检查 IP 是否被墙： 使用 ping.pe 或 boce.com 进行国内外多节点测试
  2. 如何检查 IP 纯净度： 访问 scamalytics.com 查看风险评分
  3. 如何查看端口连接数： netstat -ntu | grep :节点端口 | grep ESTABLISHED | awk '{print $5}'
  4. 推荐协议组合： VLESS + TCP + Reality + Vision，安全性高且抗检测
  5. 端口选择建议： 建议使用高位端口（40000-65000），避免常见端口被扫描

关于 XX-UI


XX-UI 是一个基于 3x-ui 二次开发的 Web 面板，支持多协议、多用户、流量统计、订阅等功能。


主要功能


  • 支持 VMess、VLESS、Trojan、Shadowsocks、Hysteria 等协议
  • 支持 Reality、TLS、WebSocket、gRPC、HTTPUpgrade 等传输方式
  • 支持订阅功能（普通订阅、JSON 订阅、Clash 订阅）
  • 支持流量统计与限制
  • 支持多用户管理
  • 支持 Telegram 机器人通知
  • 支持面板更新通知
